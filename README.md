# 工训机考复习 🛠️📚

**工程训练课程计算机考试复习工具** — 一个集知识梳理、题库展示、刷题练习和模拟考试于一体的单页 Web 应用。

> 🌐 **在线访问**: [https://hugh817.github.io/Engineering-Training-Exam-Review](https://hugh817.github.io/Engineering-Training-Exam-Review)
> 
> 🧩 **直接使用**: 浏览器打开 `index.html` 即可运行（需通过 HTTP 服务器，不可直接用 `file://` 打开）

---

## ✨ 功能一览

### 📖 知识梳理
左侧树形侧边栏包含 **14 个章节**的知识点，涵盖从材料与热处理到 CAD/CAM 的全部课程内容，支持章节展开/折叠和点击跳转。

- 材料与热处理 · 铸造 · 压力加工 · 焊接 · 切削加工基础 · 车削加工 · 钻削与镗削
- 铣削与刨削 · 磨削加工 · 钳工 · 数控与自动化 · 特种加工 · 3D打印技术 · CAD/CAM

### 🗂 题库展示
内置 **440 道题目**（判断题 + 单选题），覆盖 **13 个工种**：

| 工种 | 题量 | 工种 | 题量 |
|------|------|------|------|
| 导论/安全 | 80 | 数铣/齿 | 32 |
| 激光 | 18 | 钳工 | 33 |
| 焊接 | 20 | 数车 | 32 |
| 铸造 | 20 | WEDM | 30 |
| 热处理 | 20 | CAD | 30 |
| 车床 | 37 | VR | 29 |
| 机电 | 29 | **3D** | **30** |

支持按工种筛选、一键展开/收起答案、统计总计/判断题/单选题数量。

### 🎓 刷题 & 考试
- **📝 刷题模式** — 自由选择工种、题型（判断题/单选题）、题量，逐题作答，实时显示答案
- **📄 模拟考试** — 限时考试模式，自动判分，可查看详细答题报告
- **📕 错题本** — 自动记录答错的题目，支持逐题移除和清空

## 🚀 使用方法

### 方式一：GitHub Pages（推荐）
仓库已配置 GitHub Pages，直接访问：
```
https://hugh817.github.io/Engineering-Training-Exam-Review
```

### 方式二：本地运行
```bash
# 用 Python 启动一个简单的 HTTP 服务器
python3 -m http.server 8000
```
然后浏览器打开 `http://localhost:8000`

### 方式三：任意静态托管
将 `index.html` 上传到任何支持静态文件的托管服务（Vercel、Netlify、腾讯云静态托管等）即可直接使用。

> ⚠️ **注意**: 由于浏览器安全策略，**不能直接用 `file://` 协议**双击打开 HTML 文件。请使用 HTTP 服务器方式打开。

## 🎨 技术特色

- **纯前端** — 零依赖，零后端，一个 HTML 文件搞定全部功能
- **深色科技风** — 赛博朋克风格 UI，青色霓虹光效，毛玻璃侧边栏
- **侧边栏智能折叠** — 可折叠侧边栏，最大化内容阅读区域
- **数据全部内嵌** — 知识点内容和题库数据全部在 HTML 内，离线可用
- **响应式布局** — 自适应不同屏幕尺寸
