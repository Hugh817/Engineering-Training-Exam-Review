# 🏭 工程训练机考复习系统

> **浙江大学工程训练课程 · 机考复习平台**  
> 基于纯前端技术的交互式复习与模拟考试系统，涵盖工程训练全部核心知识模块。

---

## 📋 项目概述

本项目是为浙江大学工程训练课程机考设计的综合性复习平台，集 **知识学习** 与 **模拟考试** 功能于一体。页面采用现代化深色主题 UI，内容覆盖工程训练全部 14 个知识模块，包含 **判断题** 与 **单选题** 两大类题库，支持自主组卷、限时模拟、实时评分等完整考试流程。

### 核心功能

| 功能 | 说明 |
|------|------|
| 📖 **知识浏览** | 按章节导航，系统学习全部理论知识 |
| ✍️ **模拟考试** | 自定义题目数量，随机抽题，限时作答 |
| ✅ **实时批改** | 交卷后即时评分，展示正确答案与解析 |
| 📊 **答题统计** | 查看各章节答题情况，定位薄弱环节 |
| 🌙 **深色主题** | 护眼深色 UI，专注阅读与练习 |

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| **HTML5** | 页面结构与内容 |
| **CSS3** | 现代化深色主题 UI（变量系统、渐变背景、毛玻璃效果、流畅动画） |
| **原生 JavaScript (ES6+)** | 交互逻辑、题库引擎、考试流程控制 |
| **无外部依赖** | 纯前端实现，无需安装任何框架或库 |

### UI 设计特色

- **暗色主题**：深色背景配合柔和色彩，减轻视觉疲劳
- **毛玻璃效果**：sidebar 和导航栏使用 ackdrop-filter 实现毛玻璃质感
- **动态导航滑块**：滑动指示器跟随当前章节，过渡自然流畅
- **自适应布局**：支持侧边栏折叠，适配不同屏幕尺寸
- **视觉反馈**：答题状态、正确/错误标记清晰直观

## 🚀 使用方式

### 在线使用

直接打开 工训机考复习.html 文件即可运行，无需任何服务端环境。

### 知识学习模式

1. 左侧 **章节导航树** 选择知识点
2. 右侧展示对应章节的详细理论内容
3. 支持快速跳转和关键词定位

### 模拟考试模式

1. 在侧边栏设置 **判断题数量** 和 **单选题数量**
2. 点击 **"开始模拟考试"** 按钮
3. 系统自动从题库中随机抽题，进入限时作答
4. 完成所有题目后点击 **"提交"**
5. 系统自动批改并展示得分、正确答案与解析
